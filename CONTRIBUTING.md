# Contributing to Aevum

Thanks for your interest! Aevum is a small, self-contained front-end for
[yt-dlp](https://github.com/yt-dlp/yt-dlp) + [FFmpeg](https://ffmpeg.org).

- **Bugs / ideas:** open an issue.
- **Code:** fork the repo, make a focused change, and open a pull request. Please
  match the existing style.
- **Build from source:** see the "Build from source" section in the [README](README.md).
  You'll need Python 3.10+, plus `yt-dlp.exe` and `ffmpeg.exe` in a `bin/` folder.

Please only use Aevum for content you have the right to download.
