# Code of Conduct

Be respectful and kind. Harassment, hate speech, and personal attacks are not welcome
in this project's spaces (issues, pull requests, discussions, or anywhere else).

If someone is behaving inappropriately, please contact the maintainer via a private
report (the **Security** tab) or an issue.

This project follows the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org/).
